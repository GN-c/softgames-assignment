# Pixi Project

A PixiJS v8 demo showcasing three interactive scenes for SoftGames Developer Assignment

[Open](https://softgames-assignment-alpha.vercel.app/)

## Scenes

- Ace of Shadows

- Magic Words

- Phoenix Flame

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) in your browser.

## Scripts

| Command                  | Description                                                   |
| ------------------------ | ------------------------------------------------------------- |
| `npm run dev`            | Start development server                                      |
| `npm run build`          | Lint, type-check, and bundle for production                   |
| `npm run assetpack`      | Process raw assets into optimised bundles                     |
| `npm run download-cards` | Download Random 36 card sprites into the Ace of Shadows atlas |
